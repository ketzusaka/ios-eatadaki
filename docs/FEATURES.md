# Eatadaki Features

This document provides an overview of features in the Eatadaki app. Detailed feature documentation can be found in the `features/` subdirectory.

## Feature List

### Spots Locator
The Spots Locator feature allows users to find and select spots (restaurants, cafes, etc.) to add dish reviews to.

**Status**: Planned  
**Documentation**: [features/SPOTS_LOCATOR.md](./features/SPOTS_LOCATOR.md)

## Adding New Features

When adding a new feature:

1. Create a feature documentation file in `docs/features/`
2. Add an entry to this `FEATURES.md` file
3. Update the feature status as development progresses
4. Link to the detailed feature documentation
